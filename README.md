# **Estructura**

```
/mi-proyecto
├── /backend
│   ├── /remotion (directorio donde vive el proyecto Remotion)
│   │   ├── src
│   │   ├── package.json
│   │   ├── remotion.config.ts
│   │   ├── VideoComposition.js
│   │   └── ...otros archivos
│   ├── server.js (el servidor Express)
│   └── input.json (archivo temporal para pasar datos)
└── /frontend (proyecto React para el formulario)
```
